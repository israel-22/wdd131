About Ecuadorian Cuisine

Ecuadorian cuisine is a vibrant blend of Indigenous, Spanish, and Afro-Ecuadorian influences. Each region—Coast, Highlands, Amazon, and Galápagos—offers unique flavors, ingredients, and cooking traditions.

This website aims to introduce international visitors to authentic Ecuadorian dishes and provide easy-to-follow recipes so you can prepare these traditional meals at home.

About This Project

This website is a curated guide showcasing traditional Ecuadorian cuisine, including main dishes, desserts, and beverages from different regions of the country. The goal is to provide an accessible and visually engaging resource where users can explore authentic Ecuadorian gastronomy.

The project uses modern web development practices: semantic HTML, modular CSS, and dynamic JavaScript. All recipes are loaded from a local JSON file, which works as an internal API. This demonstrates skills in data handling, DOM manipulation, and component-based design.
